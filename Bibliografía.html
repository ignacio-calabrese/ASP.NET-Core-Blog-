Basic Concepts

To continue with a project more complex like a MVC Project remember that the ASP:NET Core separets the view, model and controller in three diferent foldiers:

  • The view files are the files with extension .cshtml which are all the user see.
  • The model files are responsable to conect your project with a DataBase.
  • The controller files are responsable to conect the model with the view.

If to all that you add Docker you can run your project inside a container making it faster and it can run in any computer no mater the Operaring System you are.
